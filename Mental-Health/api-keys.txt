API Keys Configuration
========================

To enable AI features, add your API keys here:

OPENAI_API_KEY=your_openai_api_key_here
GEMINI_API_KEY=your_gemini_api_key_here

Instructions:
1. Get OpenAI API key from: https://platform.openai.com/api-keys
2. Get Gemini API key from: https://aistudio.google.com/app/apikey
3. Replace the placeholder values above with your actual keys
4. The system will automatically use these keys for AI functionality

Note: Keep this file secure and never share your API keys publicly.
